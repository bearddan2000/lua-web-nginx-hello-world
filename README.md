# lua-web-nginx-hello-world

## Description
Embedded lua in nginx config file.

## Tech stack
- lua
- nginx

## Docker stack
- openresty/openresty:alpine

## To run
`sudo ./install.sh -u`
- Available at http://localhost

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
